# 세줄아침 콘텐츠 생성 가이드라인

## 출처 기준 (필수)
- **허용**: 언론사(조선/중앙/동아/한겨레/경향/서울신문/한국경제/매일경제 등), 통신사(연합뉴스/뉴시스/뉴스1), 전문매체(헬스조선/하이닥/코메디닷컴/약사공론 등), 정부기관(정책브리핑/복지부/국세청/금융위 등)
- **금지**: 개인 블로그(브런치/네이버 블로그/티스토리), 커뮤니티(디시/클리앙), 유튜브, SNS, 광고성 콘텐츠, 출처 불명 사이트

## 콘텐츠 작성 규칙
- title: 7~16자 명사형
- short_summary: 2~3문장, title과 겹치지 않게
- long_summary: 최소 700자, 5~8단락, \n\n 구분
- action_line: 부드러운 권유형 1문장
- 금지 표현: ~내용입니다, ~흐름입니다, ~확인해보는 게 좋습니다
